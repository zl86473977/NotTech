# 找回被扼杀的提问能力
1. 你需要具有批判性思维，用于挑战他人的假设和提出质疑
2. 提问前你需要仔细聆听
* 第一层次 - 关注自己，听的目的是根据自己的判断下结论。
* 第二层次 - 关注对方，通过聆听去支持帮助对方。
* 第三层次 - 全方位聆听，你仔细分析对方话语中的事实、感受和意图。
3. 以提问启发对方而非给予答案
4. 心怀好奇 谦逊探询
5. 学习提问技巧 刻意练习
* 开放式提问法
* 问题爆炸法

# 沟通多余问号，少用句号
* 在和他人沟通时，你要多用问号，少用句号，最好不轻易用感叹号。
## 提问的好处
1. 提问可以找出事物本质，发现问题背后的问题
2. 提问可以整理杂乱的信息，提升工作生活效率
3. 好的提问可以激发创新（提问时创新的源头，甚至可以改变世界）
4. 提问可以卖出更多的产品或服务
5. 提问可以改善人际关系
* 我很好奇，为什么你是这样想的，可以和我分享下吗
6. 提问可以让自己和他人变得更积极
